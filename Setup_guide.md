# Python Setup for Beginners

## Windows
1. Download Python:  
   ➡️ [python.org/downloads](https://www.python.org/downloads/)  
2. **CHECK THIS BOX** during install:  
   ✅ "Add Python to PATH"  
3. Verify installation:  
   - Press `Win + R`, type `cmd`  
   - Run:  
     ```bash
     python --version
     ```

## Mac/Linux
Python is pre-installed! Just verify with:  
```bash
python3 --version
